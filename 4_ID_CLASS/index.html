ID e CLASS

ID

O ID funciona como um CPF. Assim só posso ter um número de CPF, cada elemento só pode ter um valor de ID. O CPF é unico, opu seja, ninguém pode ter um CPF igual ao meu. Isso significa que o ID é unico, então não é possivel repetir o mesmo valor duas vezes

CLASS

O CLASS funciona de maneira oposota ao ID. Diferente do ID em que você só pode ter um valor, no CLASS, você pode ter vários valores. Também diferente do ID, EM QUE não se pode repetir um valor, no CLASS você pode utilizar o mesmo valor para vários elementos;
